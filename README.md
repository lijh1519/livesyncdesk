## ✨ 产品亮点

### 🚀 10秒开启协作
- 无需注册登录，打开即用
- 自动生成房间链接，分享即协作
- 支持手机、平板、电脑多端访问

### 👥 真正的实时同步
- **画布内容同步**：你画的图形，对方立刻看到
- **光标位置同步**：实时显示所有人的鼠标位置和名称
- **用户状态同步**：看到谁在线，谁加入/离开了

### 🎯 Follow Me 跟随模式
主讲人开启"Follow Me"广播，其他人自动跟随视角，非常适合：
- 远程演示和讲解
- 团队头脑风暴
- 在线教学培训

### 🛠️ 专业绘图工具
| 工具 | 功能 |
|------|------|
| ✏️ 画笔 | 自由绘制 |
| 📝 便签 | 快速记录想法 |
| ⬜ 形状 | 矩形、圆形等 |
| ➡️ 箭头 | 连接和标注 |
| ↩️ 撤销/重做 | 无限历史记录 |

### 📱 移动端友好
- 响应式设计，手机也能流畅使用
- 触控优化，手指操作无障碍

## 🚀 快速开始

### 在线使用（推荐）
直接访问：**https://livesyncdesk.vercel.app**

### 本地运行
```bash
# 1. 克隆项目
git clone https://github.com/lijh1519/livesyncdesk.git
cd livesyncdesk

# 2. 安装依赖
npm install

# 3. 配置环境变量
# 创建 .env.local 文件，添加：
# VITE_LIVEBLOCKS_PUBLIC_KEY=your_liveblocks_key

# 4. 启动项目
npm run dev
```

## 🛠️ 技术栈

- **前端框架**：React 18 + TypeScript
- **画布引擎**：tldraw 2.4
- **实时协作**：Liveblocks
- **样式方案**：Tailwind CSS 4
- **构建工具**：Vite 6
- **部署平台**：Vercel
